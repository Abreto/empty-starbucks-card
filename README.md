# empty-starbucks-card
已知每种商品的价格，给定预付卡的余额和每次最少充值金额，计算最少需要充值多少钱可以刚好把预付卡的余额花完

## example

    $ node how ./items.json 42 100

Output:

    当卡余额为 42 时，最少需充值 100 元才能恰好用完
    参考使用方案：
    消费 29 元: 浓缩咖啡一（中杯） 浓缩咖啡七（中杯） 经典巧克力饮品（中杯）

    消费 29 元: 浓缩咖啡一（中杯） 浓缩咖啡七（中杯） 经典巧克力饮品（中杯）
    消费 29 元: 浓缩咖啡一（中杯） 浓缩咖啡七（中杯） 经典巧克力饮品（中杯）
    消费 25 元: 浓缩咖啡八（中杯） 冰摇柠檬茶（中杯）
